# MusicWIKI
MusicWIKI is an unofficial Last.fm app that contains information about different music genres, the albums, artists and tracks listed under the genre.


## Features
***
The MusicWiki app provides the following features:
* Browse through different music genres.
* View information about the top albums, artists, and tracks listed under each genre.
* View details about each album, artist, and track, including album art, artist bio, and track lyrics.


## Technologies Used
***
The MusicWiki app is built using the following technologies:
* [Kotlin](https://kotlinlang.org/): A modern programming language used for developing Android apps.
* [Android Studio](https://developer.android.com/studio): An integrated development environment used for building Android apps.
* [Retrofit](https://square.github.io/retrofit/): A type-safe HTTP client used for fetching data from Last.fm's API.
* [Glide](https://github.com/bumptech/glide): An image loading and caching library for Android focused on smooth scrolling.
* [Ktlint](https://github.com/pinterest/ktlint): A Kotlin linter with built-in formatter to format the code following set of rules.
